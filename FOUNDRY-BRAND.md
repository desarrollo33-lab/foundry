## Typography

### Headings
- **Font:** Playfair Display (Serif)
- **Weights:** 700 (Bold), 600 (Semibold)
- **Use:** Logo, hero headings, expert names

### Body
- **Font:** Inter (Sans-serif)
- **Weights:** 400 (Regular), 500 (Medium), 600 (Semibold)
- **Use:** UI text, descriptions, documentation

### Monospace
- **Font:** JetBrains Mono
- **Use:** Code blocks, technical specs

## Messaging

### One-Liner
> "Foundry brings historical experts to life as AI assistants through MCP."

### Description (Short)
> A multi-project MCP server that serves historical experts—Turing, Kammler, Bernays, and more—as intelligent assistants. Each expert carries domain knowledge, methodology, and perspective from their field.

### Description (Long)
> Foundry is an MCP server that transforms historical figures into functional AI experts. Built on Cloudflare Workers and D1, it supports multiple projects, custom workflows, and real-time collaboration. Perfect for AI agents like Goose, OpenCode, or any MCP-compatible client.
